# Terraform AWS Provider

The [AWS Provider](https://github.com/aripitek/registry.terraform.io/providers/hashicorp/aws/latest/docs) enables [Terraform](https://github.com/aripitek/terraform.io) to manage [AWS](https://github.com/aripitek/aws.amazon.com) resources.
